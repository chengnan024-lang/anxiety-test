# 焦虑程度测试小程序

基于GAD-7与PHQ-9国际权威量表的心理学测评H5小程序。

## 功能

- 免费测评：8题快速版、15题深度版
- 付费完整版（¥2.99）：三维雷达图、7天行动计划、精神分析深度解读
- 危机干预：自伤念头检测与心理援助热线

## 在线体验

部署后访问：`https://[你的用户名].github.io/anxiety-test/`

## 技术栈

- 纯HTML/CSS/JavaScript
- Canvas绘制雷达图
- 无需后端

## 部署

```bash
# 1. 克隆仓库
git clone https://github.com/[你的用户名]/anxiety-test.git
cd anxiety-test

# 2. 推送代码
git add .
git commit -m "Initial commit"
git push origin main

# 3. 开启GitHub Pages
# 在仓库Settings > Pages中，Source选择"main branch"
```
